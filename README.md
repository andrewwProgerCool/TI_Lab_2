# TI_LAB_2

Веб-приложение на `Vue 3 + Vite` для шифрования и расшифровки файлов с помощью `XOR` и 32-битного бинарного ключа.

Интерфейс позволяет:
- ввести ключ вручную;
- сгенерировать случайный 32-битный ключ;
- загрузить исходный файл;
- выполнить `encrypt` или `decrypt`;
- посмотреть превью входных и выходных данных;
- скачать результат;
- увидеть количество выполненных сдвигов ключа.

## Что реализовано

- 2 режима работы: `Encrypt` и `Decrypt`
- 32-битный ключ
- визуализация ключа в виде 32 ячеек
- подсветка отводов `1`, `27`, `28`, `32`
- обработка файлов через `XOR`
- поблочный сдвиг ключа в процессе обработки
- генерация случайного бинарного ключа
- просмотр результата в `HEX` или текстовом виде
- скачивание обработанного файла

## Как работает ключ

Поле ключа принимает любой текст, но в алгоритме используются только символы `0` и `1`.

Правила:
- из введённого текста извлекаются только `0/1`;
- используется максимум `32` бинарных символа;
- как только набрано `32` бит, новые бинарные символы больше не добавляются;
- небинарные символы можно удалять и редактировать, но в шифровании они не участвуют.

Ключ разбивается на 4 байта по 8 бит и используется при `XOR`-обработке файла. Во время работы приложения ведётся счётчик сдвигов ключа.

## Алгоритм обработки

Текущая реализация работает по такой схеме:

1. Пользователь вводит бинарный ключ длиной `32` бита.
2. Ключ разбивается на 4 байта.
3. Файл читается как последовательность байтов.
4. Каждый байт файла обрабатывается через `XOR` с текущим байтом ключа.
5. После обработки очередного блока ключ сдвигается.
6. Новый бит вычисляется по отводам `1`, `27`, `28`, `32`.
7. Результат собирается в новый бинарный файл и доступен для скачивания.

Так как используется `XOR`, одна и та же операция с тем же ключом позволяет и зашифровать, и расшифровать данные.

## Поведение интерфейса

- В блоке `KEY` отображается текущее количество заполненных битов.
- Кнопка `Генерация` создаёт случайный 32-битный ключ.
- В блоке `IN` выбирается исходный файл.
- В блоке `OUT` появляется результат обработки и ссылка на скачивание.
- Под кнопкой запуска выводится количество сдвигов ключа.

## Форматы предпросмотра

Приложение старается показывать результат удобно:
- если данные похожи на текст, выводится текстовый фрагмент;
- если данные бинарные, показывается `HEX`-предпросмотр.
